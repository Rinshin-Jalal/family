## TECHNICAL REQUIREMENTS

### Backend
1. OCR service for handwritten text recognition
2. Document parsing (PDF, TXT, DOCX)
3. Voice cloning integration (ElevenLabs API or similar)
4. Text-to-speech with voice cloning

### iOS
1. Camera integration for photo capture
2. Document picker for file upload
3. Photo library access
4. OCR scanning UI

### Database Updates
```sql
-- Add collection method to stories table
ALTER TABLE stories ADD COLUMN collection_method VARCHAR(20);
-- values: 'audio', 'written', 'photo', 'document', 'voice_clone'

-- Add voice clone metadata
ALTER TABLE stories ADD COLUMN is_voice_cloned BOOLEAN DEFAULT FALSE;
ALTER TABLE stories ADD COLUMN original_voice_sample_url TEXT;
```
